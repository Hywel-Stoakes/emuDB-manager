Instructions

  • Upload a ZIP file here.
  • The ZIP file must contain a valid EMU speech database: a directory called name_emuDB, which in turn contains name_DBconfig.json and a number of session directories.
  • The EMU speech database can then either be saved as a new database or merged with an existing database.
  • If you want to merge two databases, you have to make sure the new data and the old data fit together.
    • The new database must not contain sessions that are already there in the old database.
    • It is important, that the annotation structure of both databases is the same. They must contain the same levels and the same types of links.
