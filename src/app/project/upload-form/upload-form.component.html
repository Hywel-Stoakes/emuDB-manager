Instructions

  • You can either upload speech databases here or new data to add them to an existing database.
  • If you have a complete EMU speech database, simply “zip it up” and upload the zip file here.
    • You’re done :-)
  • If you have new sessions that you want to add to an EMU speech database you have already uploaded here, you have to make sure the new data and the old data fit together.
    • Your new data will most likely be an EMU speech database already, that is, you have a number of folders called *_ses along with a file called <databasename>_DBconfig.json in a folder called <databasename>_emuDB. It does not matter whether the old and the new speech database have the same name.
    • It is important, however, that the annotation structure is the same. The bundles in both old and new data must contain the same levels and the same types of links.
